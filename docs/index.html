Overview

Open-xtract is an open-source framework that extracts structured data from PDFs. Designed with modularity in mind, it allows you to bring your own OCR or LLM models and extend to any file type.

Model Agnostic

Simple adapter API works with any OCR engine or large language model. Swap models at will without changing your code.

PDF-First Ingestion

Layout-aware parsing produces clean, tokenized text that preserves document structure and formatting.

Cited Retrieval

Vector search with reranked answers and inline citations. Every chunk is embedded and served via RAG.

Installation

Install open-xtract using pip:

pip install open-xtract

Requirements

  • Python 3.12 or higher
  • Compatible with all major operating systems
